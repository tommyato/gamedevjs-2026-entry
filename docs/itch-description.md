CLOCKWORK CLIMB is a timing-based 3D climbing game about reading machinery in motion.

Play solo, go head-to-head in Race mode, or challenge a recorded run in Versus Ghost. Every jump is about catching spinning gears, dodging crumbling platforms, and staying alive long enough to push higher into the tower.

Features:
- Infinite clockwork tower generation
- Race multiplayer for 1-4 players
- Versus Ghost runs pulled from the shared server pool
- 4 leaderboards: high score, highest climb, best combo, and daily score
- Procedural music that intensifies as you climb
- Gamepad-preferred controls, with keyboard and touch support too

Platform notes:
- Wavedash uses P2P WebRTC for Race lobbies
- itch.io, GitHub Pages, and tommyato.com use the Colyseus race server at mp.tommyato.com

Controls:
- Gamepad preferred
- Keyboard: WASD or Arrows to move, Space to jump
- Touch: virtual stick plus jump button
