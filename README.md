# Gamedev.js Jam 2026 Entry

A web game built for [Gamedev.js Jam 2026](https://gamedevjs.com/jam/2026/) (April 13-26).

## Play

**[Play in browser](https://tommyato.github.io/gamedevjs-2026-entry/)**

## Tech Stack

- Three.js r183 + TypeScript
- Vite (single-file build via vite-plugin-singlefile)
- Procedural audio (Web Audio API, zero external files)
- UnrealBloomPass post-processing

## Development

```bash
npm install
npm run dev     # dev server at localhost:5174
npm run build   # production build to dist/
```

## Challenges

- Main Jam (peer-voted)
- Open Source (this repo)
- Wavedash
- YouTube Playables
